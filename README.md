# Mariana Portfolio

Personal portfolio for Mariana Djum, built with Next.js App Router and Tailwind CSS.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Requirements

- Node.js 22 LTS

## Scripts

- `npm run dev` - start development server
- `npm run lint` - run ESLint
- `npm run build` - create production build
- `npm run start` - run production server

## Project Structure

- `app/` - layout, page composition, and global styles
- `components/` - UI sections and reusable cards
- `content/` - portfolio/profile/workflow content data
- `lib/` - local data utilities
